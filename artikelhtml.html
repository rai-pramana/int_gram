Pendahuluan HTML

HTML merupakan bahasa pemrograman yang menggunakan penanda (markup) untuk membuat dan mendeskripsikan struktur dari suatu halaman web.

Penanda dalam dokumen HTML disebut sebagai tag. Tag HTML selalu berpasangan (tag pembuka dan tag penutup), namun ada beberapa tag yang tidak memiliki tag penutup, yaitu tag <br>, <hr>, <img>, dll.
Penulisan tag penutup ditambahkan tanda garis miring (/) di depan nama tag tersebut. Contohnya yaitu sebagai berikut. 

<p></p>

Komponen penyusun dokumen HTML yang diapit oleh tag disebut sebagai elemen. Elemen dapat berisi elemen lainnya (nested element). Contohnya yaitu sebagai berikut. 

<p align="left">Halo</p>

Pengubah (modifier) perilaku dari suatu elemen dalam dokumen HTML disebut sebagai atribut. Atribut biasanya terdapat pada tag pembuka. Pada contoh sebelumnya, atribut align merupakan atribut yang digunakan untuk mengatur perataan teks.


Peralatan untuk Menggunakan HTML

  1. Teks Editor

    2. Teks editor digunakan untuk membuat dan menulis dokumen HTML. Contohnya yaitu Visual Studio Code, Notepad, Notepad++, Sublime Text, dll.

  2. Web Browser

    3. Web browser digunakan untuk menampilkan (render) dokumen HTML. Contohnya yaitu Google Chrome, Firefox, Microsoft Edge, dll.


Struktur Dasar HTML

<!DOCTYPE html>
<html lang="en">
<head>
    <title>Judul Web</title>
</head>
<body>
    <!--Komentar-->
</body>
</html>

  1. Bagian Deklarasi

    2. Bagian deklarasi ditandai dengan tag <!DOCTYPE html> yang menyatakan bahwa dokumen ini bertipe HTML dan versinya adalah HTML 5.
    Selanjutnya, tag pembuka HTML <html> yang digunakan untuk membungkus tag <head> dan tag <body>, serta tag HTML lainnya, lalu diakhiri dengan tag penutupnya yaitu </html>.
    Atribut lang="en menyatakan bahwa dokumen HTML menggunakan bahasa Inggris.

  2. Bagian Head

    3. Bagian head biasanya digunakan untuk menuliskan tag <title> untuk judul halaman web, serta untuk menuliskan kode CSS dan Javascript.

  3. Bagian Body

    4. Bagian body merupakan bagian isi/konten dari dokumen HTML yang akan ditampilkan pada halaman web pada browser. Bagian ini dapat disisipi dengan berbagai macam tag yang digunakan untuk menuliskan konten (teks, gambar, video, dan link) pada dokumen HTML.

  4. Komentar

    5. Komentar merupakan bagian dari dokumen HTML yang tidak akan ditampilkan pada halaman web, yang digunakan untuk menambahkan penjelasan terkait dokumen HTML yang dibuat. Penulisan komentar menggunakan tag <!-- komentar -->.

Pembuatan Heading

Heading merupakan judul yang ditambahkan pada artikel atau halaman web. Pembuatan judul pada dokumen HTML menggunakan tag <h1> (judul level pertama yang berukuran paling besar) hingga <h6> (sub judul yang berukuran paling kecil). 

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

Tampilan pada halaman web browser.

Pemformatan Teks

Pemformatan teks bertujuan agar suatu teks menjadi lebih mudah dibaca karena terdapat penegasan, seperti teks tebal, miring, garis bawah, tercoret, pemangkatan, pewarnaan, dll.

Tag <font> merupakan tag yang memiliki berbagai atribut untuk memformat teks, yaitu sebagai berikut.

  • <font color=”value”> untuk mewarnai teks. Value yang dapat dimasukkan yaitu nama warna atau bilangan heksadesimal (#000000 hingga #ffffff). 

    <font color="#8400ff">Lorem ipsum dolor sit amet consectetur adipisicing elit.</font>
<br>
<font color="red">Lorem ipsum dolor sit amet consectetur adipisicing elit.</font>

    Tampilan pada halaman web browser.


  • <font size=value> untuk menentukan ukuran font. Value yang dapat dimasukkan yaitu bilangan dari 1 hingga 7. 

    <font size="1">Lorem ipsum dolor sit amet consectetur adipisicing elit.</font>
<br>
<font size="2">Lorem ipsum dolor sit amet consectetur adipisicing elit.</font>
<br>
<font size="3">Lorem ipsum dolor sit amet consectetur adipisicing elit.</font>
<br>
<font size="4">Lorem ipsum dolor sit amet consectetur adipisicing elit.</font>

<font size="5">Lorem ipsum dolor sit amet consectetur adipisicing elit.</font>
<br>
<font size="6">Lorem ipsum dolor sit amet consectetur adipisicing elit.</font>
<br>
<font size="7">Lorem ipsum dolor sit amet consectetur adipisicing elit.</font>

    Tampilan pada halaman web browser.


  • <font face=”value”> untuk menentukan jenis font. 

    <font face="Arial">Lorem ipsum dolor sit amet consectetur adipisicing elit.</font>
<br>
<font face="Courier New">Lorem ipsum dolor sit amet consectetur adipisicing elit.</font>

    Tampilan pada halaman web browser.

Pembuatan List

List merupakan format yang digunakan untuk merepresentasikan suatu daftar. Terdapat tiga jenis list pada HTML, yaitu sebagai berikut.

  1. Unordered List

    2. Unordered list merupakan daftar yang tidak mengutamakan urutan dari setiap itemnya. Tag yang digunakan untuk membuat unordered list adalah tag <ul>, serta tag <li> (list item) untuk menyatakan daftar itemnya.

    Atribut type digunakan untuk mengganti simbol unordered list, dengan nilai sebagai berikut.

    • square untuk simbol persegi.
    • disc (default) untuk simbol piringan.
    • none tidak memakai simbol.
    • circle untuk simbol lingkaran. 

    List Barang:
<ul type="square">
    <li>Buku</li>
    <li>Pensil</li>
    <li>Penggaris</li>
</ul>

    Tampilan pada halaman web browser.


  2. Ordered List

    3. Ordered list merupakan daftar yang mengutamakan urutan dari setiap itemnya. Tag yang digunakan untuk membuat ordered list adalah tag <ol>, serta tag <li> (list item) untuk menyatakan daftar itemnya.

    Atribut type digunakan untuk mengganti simbol ordered list, dengan nilai sebagai berikut.

    • a untuk alfabet (a, b, c, dst.).
    • A untuk alfabet (A, B, C, dst.).
    • 1 (default) untuk bilangan (1, 2, 3, dst.).
    • i untuk bilangan romawi (i, ii, iii, dst.).
    • I untuk bilangan romawi kapital (I, II, III, dst.). 

    List Kegiatan:
    <ol type="I">
    <li>Coding</li>
    <li>Gaming</li>
    <li>Sport</li>
</ol>

    Tampilan pada halaman web browser.


  3. Description/Definition List

    4. Description/definition list merupakan daftar yang berisi penjelasan dari sesuatu. Tag yang digunakan untuk membuat description list adalah tag <dl>, tag <dt> (description term) untuk menyatakan istilah yang akan dijelaskan, serta tag <dd> (description description) untuk membuat penjelasan dari kata tersebut. 

    <dl>
    <dl>Mouse</dl>
    <dd>Periferal untuk menggerakkan pointer pada komputer</dd>

    <dl>Keyboard</dl>
    <dd>Periferal untuk menginput karakter pada komputer</dd>
</dl>

    Tampilan pada halaman web browser.


Nested List (List Bersarang)

List dapat dibuat di dalam list lainnya. 

List kegiatan:
<ol type="I">
    <li>Coding
        <ul type="circle">
            <li>HTML</li>
            <li>CSS</li>
            <li>JS</li>
        </ul>
    </li>
    <li>Gaming</li>
        <ol>
            <li>Genshin Impact</li>
            <li>Flow Free</li>
        </ol>
    <li>Sport</li>
</ol>

Tampilan pada halaman web browser.

Penambahan Hyperlink

Hyperlink atau link merupakan elemen HTML yang digunakan untuk menghubungkan halaman web yang satu dengan halaman web lainnya, mengunduh file, menjalankan fungsi Javascript, scroll halaman web, dll.

Pembuatan link pada HTML menggunakan tag <a> dengan atribut wajib yaitu href untuk menyimpan alamat tujuan (URL) dari link tersebut.

Pada halaman web, link akan secara default ditandai dengan warna biru dan garis bawah.

Jenis-jenis Link pada HTML yaitu sebagai berikut.

  1. Internal Link

    2. Internal link merupakan link yang memiliki alamat tujuan yaitu domain atau halaman web itu sendiri. 

    <a href="C:\Users\raipr\Desktop\biodata.html">Biodata</a>

    Tampilan pada halaman web browser.


  2. Eksternal Link

    3. Eksternal link merupakan link yang memiliki alamat tujuan yaitu domain lain. 

    <a href="https://www.youtube.com/">Youtube</a>

    Tampilan pada halaman web browser.


Atribut target digunakan menentukan target pembukaan link tersebut. Nilai yang dimasukkan yaitu sebagai berikut.

  • _blank digunakan untuk membuka link pada jendela/tab baru. 

    <a href="https://www.youtube.com/" target="_blank">Youtube</a>

    Contoh: Youtube


  • _self digunakan untuk membuka link pada halaman itu sendiri (default). 

    <a href="https://www.youtube.com/" target="_self">Youtube</a>

    Contoh: Youtube


  • _top digunakan untuk menuju bagian paling atas pada halaman web. 

    <a href="#" target="_top">Menuju ke Atas</a>

    Contoh: Menuju ke Atas


  • nama-frame digunakan untuk membuka link pada tag <iframe> dengan nama tertentu. 

    <a href="C:\Users\raipr\Desktop\biodata.html" target="frameA">Link Frame</a>
<iframe name="frameA" width="1000px" height="600px"></iframe>

    Tampilan pada halaman web browser.


Atribut title digunakan untuk membuat informasi tambahan saat link disentuh pointer (PC) atau menahan tap (Smartphone). 

<a href="" title="Hover link">Link</a>

Contoh: Link


Link anchor merupakan link yang digunakan untuk menuju ke suatu elemen pada dokumen HTML. Pembuatan link anchor pada dokumen HTML menggunakan tanda pagar (#), yang diikuti dengan nama id dari elemen tujuan. 

<a href="#pendahuluan">Menuju ke Pendahuluan</a>

Contoh: Menuju ke Pendahuluan
